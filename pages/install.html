Reg Search 1.0

Basic Use

Open the plugin and type in a regex query. Hit Enter to search. Use the Next and Prev buttons to iterate through your search results. Hit the Copy button to copy the search results as JSON array to the clip board. Modifying the search query will clear the current search. Note that this extension will not work on the Chrome store, so do not test it on any Chrome store pages.

Short Cut Action
Alt+Shift+F open search plug-in
Alt+N cycle next result
Alt+P cycle previous result
Alt+C copy search results as JSON array to clip board
Notes and Hints

You can set the default case sensitivity on the options page This extension is open source and the code can be found on github. If you find any bugs or have suggestions for improvements, let us know. This extension is published under the MIT license.
